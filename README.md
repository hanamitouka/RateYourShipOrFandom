# RateYourShipOrFandom
锐评你的家产/锐评你圈。基于使用电脑和ai打黑工，带你细数家产（圈子）曾经或现在的辉煌（冷清）（冷圈人悲伤）

# 使用相关：

跑那个main.py的文件就好了，输入想要查找的同人圈的网址，就会自动分析。

为了防止ao3网站或者电脑死掉，对于大火的文章太多的圈子，不会采用全样本分析，而是会抽取20页，即400篇文章分析。但是准确性还可以。

能够输出的信息：

角色、关系、同人圈、分类、警告信息、评级、自由标签、字数、点击量、kudos、评论之类的
